templates - все шаблоны должны храниться именно здесь, т.к фласк по умолчанию ищет их в папке templates, которую мы должны создать
static - тут хранятся статические файлы (картинки, css, js)

{% block title %}{% endblock %} - Если говорить в кратце, то это шаблонизатор, который позволяет менять html при помощи шаблонов (в нашем случае base.html - шаблон, а остальные html - это данные, которые имеют свою уникальную информацию, но стиль/оформление у них будет идентично шаблону(если его не менять)).
block - это начало блока. endblock - конец блока. extends - подключение шаблона ({% extends 'base.html' %}). title - название и оно может быть любым



Если вдруг мы пропишем новые стили, а они не обновляются, то это значит, что стили попали в Кэш. Чтобы очистить Кэш нужно: Нажать пкм на сайте -> Просмотреть Код -> Зажать кнопку перезагрузки браузера -> Очистка кэша и жёсткая перезагрузка
